# Calculadora de IMC

Aplicativo mobile simples desenvolvido com **React Native** para calcular o **Índice de Massa Corporal (IMC)**.

O projeto foi desenvolvido como exercício prático durante uma aula com o professor **Hercules Ramos**, com o objetivo de praticar lógica de programação e conceitos básicos de desenvolvimento mobile.

## 🚀 Funcionalidades

- Inserir **peso (kg)**
- Inserir **altura (m)**
- Calcular automaticamente o **IMC**
- Exibir a **classificação do resultado**:
  - Abaixo do peso
  - Normal
  - Sobrepeso
  - Obesidade

## 🧠 Lógica do Projeto

O aplicativo possui uma função responsável por:

1. Receber o **peso** e a **altura** do usuário.
2. Calcular o IMC
## 🎨 Design

O design da interface foi prototipado no **Figma**, buscando uma interface simples, clara e fácil de usar.

## 🛠️ Tecnologias utilizadas

- React Native
- JavaScript
- StyleSheet (CSS-in-JS do React Native)
- Figma
